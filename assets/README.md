# assets

Statičke datoteke koje poslužuje Vercel (dostupne na `/assets/...`).

## Logotip hrvatskog turizma

Ovdje ide logotip hrvatskog turizma za zaglavlje. Datoteku dodajte kao:

- **`hrvatski-turizam-logo.png`** (ili `.svg` / `.jpg`)

Zaglavlje (`index.html`) povezuje se na `/assets/hrvatski-turizam-logo.<ekstenzija>`.
Ako je ekstenzija drukčija od `.png`, javite pa se `src` uskladi.
